# Frontend (Vue 3 + Vite)

Este folder contiene el **frontend** de la app.

## Requisitos

- Node.js (recomendado 18+)

## Instalación

Desde este directorio:

- `npm install`

## Scripts

- `npm run dev` (levanta Vite en http://localhost:5173)
- `npm run build`
- `npm run lint`
- `npm run typecheck`
- `npm run test` (watch)
- `npm run test:run` (CI)

## Nota

El backend corre en `http://localhost:3001` y el cliente se conecta vía Socket.IO.

Para más contexto de arquitectura, eventos y setup completo, revisa el README principal del repo: [../README.md](../README.md)
